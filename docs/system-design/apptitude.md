## Important Formulas (Quick Recall)

### üìà Percentages & Growth

* **% = (Value / Total) √ó 100**
* **Growth % = ((New ‚Äì Old) / Old) √ó 100**
* **Successive % Change = a + b + (ab / 100)**
* **Revenue = Price √ó Quantity**
* **Profit Margin = (Profit / Revenue) √ó 100**

### üí∞ Profit, Loss & Discount

* **Profit = SP ‚Äì CP**, **Loss = CP ‚Äì SP**
* **Profit % = (Profit / CP) √ó 100**
* **Selling Price = CP √ó (1 + Profit% / 100)**
* **Discount % = (Discount / MP) √ó 100**

### ‚öñÔ∏è Ratio & Proportion

* **a : b = a/b**, and if **a : b = c : d ‚Üí ad = bc**
* **Share Formula:** (m / (m + n)) √ó Total
* **Weighted Average:** (Œ£ Quantity √ó Value) / Total Quantity

### üïí Time, Work & Distance

* **Work = Rate √ó Time**
* **If A finishes in x days ‚Üí 1/x work per day**
  Combined (A + B) = (1/x + 1/y)
* **Speed = Distance / Time**
* **Average Speed = (2xy) / (x + y)**

### üíπ Interest & Investment

* **SI = (P √ó R √ó T)/100**
* **CI = P(1 + R/100)^T ‚Äì P**
* **ROI = (Gain or Loss / Investment) √ó 100**
